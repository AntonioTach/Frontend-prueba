PRUEBA TÉCNICA YO CONTIGO Presiona para ver la descripción de la prueba PRUEBA TÉCNICA YO CONTIGO API en Node.js:
  • Crea una API en Node.js conectada a una base de datos SQL.
  • Crea una ruta GET que devuelva una lista de usuarios en formato JSON que contenga nombre, correo electrónico, edad y sexo (puedes utilizar datos de ejemplo).
  • Crea una ruta POST para crear nuevos usuarios.
  • Crea una ruta PUT para editar los datos del usuario.
  • Crea una ruta DELETE que borre al usuario correspondiente.
Proyecto Angular:
  • Crea un nuevo proyecto de Angular. Asegúrate de tener Angular CLI instalado.
  • En la carpeta del proyecto Angular, crea un componente que mostrará una lista de usuarios. Debes tener un método para realizar la solicitud HTTP a la API y mostrar los usuarios en una lista.
  • En el componente creado, muestra los usuarios en una tabla HTML con las siguientes columnas: ID, nombre, correo electrónico, edad, sexo.
  • Genera un nuevo componente e implementa un formulario HTML que permita al usuario ingresar los datos de un nuevo usuario (nombre, correo electrónico, edad, sexo) y enviarlos a él.
  • Implementa la funcionalidad para manejar errores en caso de que las solicitudes HTTP hacia el API. Muestra un mensaje de error en la interfaz de usuario en caso de que ocurra un error.
Presentación:
  • El proyecto terminado (backend y frontend) deberás subirlo a un repositorio en GitHub para poder revisar el código utilizado.
  • Si tienes un servidor disponible donde desplegar la aplicación, ejecuta el servidor y agrega el enlace.
  • Si no tienes un servidor disponible, utiliza ngrok para generar una URL pública temporal que nos permita ver la aplicación en un entorno de desarrollo.
